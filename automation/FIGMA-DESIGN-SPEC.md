# Figma Design Specifications

Design guidelines and asset specifications for U.S. COVID Vaccine Injuries social media and marketing materials.

## Brand Identity

### Colors

| Name | Hex | Usage |
|------|-----|-------|
| Primary Blue | `#1e3a5f` | Headers, primary buttons |
| Secondary Blue | `#2c5282` | Accents, links |
| Accent Gold | `#d69e2e` | Highlights, CTAs |
| Success Green | `#38a169` | Positive stats, success states |
| Alert Red | `#e53e3e` | Warnings, critical stats |
| Background Light | `#f7fafc` | Page backgrounds |
| Text Primary | `#1a202c` | Body text |
| Text Secondary | `#718096` | Captions, muted text |

### Typography

| Element | Font | Weight | Size |
|---------|------|--------|------|
| Headlines | System UI / Inter | Bold (700) | 32-48px |
| Subheadlines | System UI / Inter | Semibold (600) | 24-28px |
| Body | System UI / Inter | Regular (400) | 16-18px |
| Captions | System UI / Inter | Regular (400) | 14px |
| Stats/Numbers | System UI / Inter | Bold (700) | 48-72px |

### Logo Usage

- Primary: Scale icon + "U.S. Covid Vaccine Injuries" text
- Icon only: Scale icon (⚖️) for small spaces
- Minimum size: 24px icon, 120px full logo

- Source citation small

## Video Thumbnails

### YouTube Thumbnail

- Dimensions: 1280x720px
- Include: Face (if testimonial), large text, bright colors
- Text: Max 5 words, readable at small size
- Style: High contrast, emotion-evoking

### Video Overlay Elements

- Lower third: Name + title bar
- End screen: CTA + subscribe button
- Watermark: Small logo, corner placement

---

## Print Materials

### Flyer (Letter Size)

- Dimensions: 8.5x11in (2550x3300px at 300dpi)
- Bleed: 0.125in
- Safe zone: 0.5in from edges
- Content: QR code to website, key stats, contact info

### Business Card

- Dimensions: 3.5x2in (1050x600px at 300dpi)
- Front: Logo, website, tagline
- Back: QR code, key stat

## Export Settings

### Web/Social (PNG)

- Format: PNG-24
- Resolution: 1x (actual pixels)
- Color profile: sRGB

### Web/Social (JPG)

- Format: JPEG
- Quality: 80-90%
- Resolution: 1x
- Use for: Photos, complex images

### Print (PDF)

- Format: PDF
- Color profile: CMYK
- Include bleed marks
- Flatten transparency
